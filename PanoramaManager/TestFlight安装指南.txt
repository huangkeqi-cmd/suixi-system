================================================================================
TestFlight 安装指南
================================================================================

一、通过 TestFlight 安装（推荐）
--------------------------------

1. 获取 TestFlight 邀请链接
   - 请联系项目管理员获取 TestFlight 公开测试链接
   - 链接格式: https://testflight.apple.com/join/XXXXXXXX

2. 安装 TestFlight App
   - 在 App Store 搜索 "TestFlight" 并安装
   - 或访问: https://apps.apple.com/cn/app/testflight/id899247664

3. 加入测试
   - 点击管理员提供的邀请链接
   - 系统会自动跳转到 TestFlight App
   - 点击"接受"加入测试

4. 安装应用
   - 在 TestFlight 中找到"随系·影像管理器"
   - 点击"安装"按钮
   - 等待下载完成

5. 开始使用
   - 安装完成后，桌面会出现"随系·影像管理器"图标
   - 首次打开时需要授权相机、相册、蓝牙权限

================================================================================
二、通过 Xcode 安装（开发者）
--------------------------------

1. 准备工作
   - macOS 14.0+
   - Xcode 15.0+
   - Apple Developer 账号（免费或付费均可）
   - iPhone / iPad（iOS 16.0+）

2. 打开项目
   - 解压交付包中的 iOS 源码
   - 双击 PanoramaManager.xcodeproj 用 Xcode 打开

3. 配置签名
   - 在 Xcode 左侧栏选中项目
   - 选择 TARGETS -> PanoramaManager
   - 进入 Signing & Capabilities 标签
   - 在 Team 下拉框中选择你的 Apple ID
   - 如果没有，点击"Add Account..."添加

4. 连接设备
   - 用数据线连接 iPhone/iPad 到 Mac
   - 在 Xcode 顶部工具栏选择你的设备

5. 编译运行
   - 点击左上角的"运行"按钮（▶️）
   - 首次运行需要在设备上信任开发者证书
   - 设置 -> 通用 -> VPN与设备管理 -> 信任 [你的Apple ID]

6. 安装完成
   - 应用会自动安装到设备上
   - 可以断开数据线继续使用

三、常见问题
--------------------------------

Q1: TestFlight 提示"此 Beta 版本目前不接受任何新测试员"
A: 请联系管理员增加测试名额，或等待下一个版本发布。

Q2: Xcode 编译报错"Signing for PanoramaManager requires a development team"
A: 需要选择或添加 Apple Developer 账号。免费账号可以真机调试，有效期7天。

Q3: 应用打开后闪退
A: 1. 检查 iOS 版本是否 ≥ 16.0
   2. 检查是否已授权所有必要权限（相机、蓝牙）
   3. 重启设备后重试

Q4: 蓝牙无法连接相机
A: 1. 确认相机已开机并进入配对模式
   2. 检查 iOS 设置中蓝牙是否开启
   3. 确认已在弹窗中授权蓝牙权限
   4. 使用手动输入模式作为备用方案

Q5: 如何更新到最新版本？
A: TestFlight 用户会收到自动更新通知，点击更新即可。
   Xcode 用户需要重新拉取代码并编译运行。
