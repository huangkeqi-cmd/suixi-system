================================================================================
影像管理器 - 使用说明
================================================================================

一、系统简介
-----------
本系统是一套用于商业改造现场的 100% 离线影像管理工具，包含：

1. iOS App (手机端) - 现场拍摄影像并在平面图上标记位置
2. PC 管理器 (电脑端) - 导入影像、关联标记点、生成网页查看器
3. 网页查看器 - 在浏览器中浏览，支持手机扫码访问

================================================================================
二、工作流程
================================================================================

步骤 1: 手机端创建项目
----------------------
1. 打开 iOS App "随系·影像管理器"
2. 点击"新建项目"，输入项目名称
3. 导入平面图（支持 JPG/PNG/PDF）
4. 在平面图上标记拍摄点位

步骤 2: 现场拍摄
----------------
1. 连接 DJI Osmo 360 相机（蓝牙）
2. 点击标记点，按快门拍摄
3. 相机会自动将照片保存到 SD 卡
4. 拍摄完成后导出项目（生成 ZIP 文件）

步骤 3: PC 端导入和关联
------------------------
1. 打开 "影像管理器.exe"
2. 导入手机端导出的 ZIP 文件
3. 选择 SD 卡照片文件夹进行智能匹配
4. 系统会自动根据文件名关联照片和标记点
5. 手动确认相似匹配和缺失的照片

步骤 4: 生成网页查看器
----------------------
1. 点击"生成本地网页"按钮
2. 系统会自动创建 viewer/ 文件夹
3. 点击"打开本地网页"启动服务器
4. 用浏览器访问，或扫描二维码在手机上查看

================================================================================
三、iOS App 使用详解
================================================================================

3.1 项目列表
-----------
- 显示所有项目，按修改时间倒序排列
- 左滑项目可删除

3.2 新建项目
-----------
- 项目名称: 最多 50 个字符，不能包含特殊符号
- 导入平面图:
  * JPG/PNG: 直接使用
  * PDF: 自动转换为首页图片（宽度 2048px）

3.3 标记点操作
-------------
- 添加标记: 长按平面图 0.3 秒
- 移动标记: 选中后拖动
- 删除标记: 长按标记 0.5 秒，确认删除
- 状态颜色:
  * 灰色空心: 仅标记，未拍摄
  * 蓝色实心: 已拍摄，未导入 PC
  * 绿色实心: 已在 PC 端关联照片

3.4 蓝牙拍摄
-----------
- 点击顶部"连接相机"按钮
- 选择 DJI Osmo 360 设备进行连接
- 连接成功后显示电量
- 点击标记点，按快门拍摄
- 拍摄成功后标记点变蓝

3.5 手动输入模式（备用）
-----------------------
如果蓝牙无法获取文件名：
1. 正常拍摄照片
2. 在 App 中点击标记点
3. 选择"手动输入文件名"
4. 输入 SD 卡中显示的文件名（如 DJI_20250122_143015.JPG）

3.6 导出项目
-----------
- 点击"导出项目"按钮
- 选择分享方式（AirDrop、微信、保存到文件）
- 生成的 ZIP 文件包含项目配置和平面图

================================================================================
四、PC 管理器使用详解
================================================================================

4.1 导入项目
-----------
- 点击"导入项目"
- 选择手机端导出的 ZIP 文件
- 选择解压目录
- 系统会自动校验项目文件

4.2 导入影像
---------------
- 点击"导入影像"
- 选择 SD 卡中存放照片的文件夹
- 系统会递归查找所有 JPG/JPEG 文件
- 智能匹配策略:
  1. 精确匹配: 文件名完全一致
  2. 相似匹配: 时间戳相差 ≤ 5 秒
  3. 手动指定: 用户手动选择文件

4.3 编辑标记点
-------------
- 点击平面图上的标记点
- 右侧面板显示详细信息
- 可编辑:
  * 自定义名称
  * 在文件夹中显示照片
  * 更换关联的照片
  * 删除标记点
- 支持在平面上拖拽移动位置

4.4 生成网页查看器
-----------------
- 点击"生成本地网页"按钮
- 选择输出目录
- 系统会创建包含以下内容的 viewer 文件夹:
  * index.html - 主页面
  * pannellum/ - 全景渲染引擎
  * data/ - 项目数据
  * photos/ - 影像文件

4.5 启动本地服务器
-----------------
- 点击"打开本地网页"按钮
- 系统会自动:
  1. 启动 HTTP 服务器（端口 8080 或自动分配）
  2. 获取本机局域网 IP
  3. 用浏览器打开本地地址
  4. 生成二维码供手机扫描

- 访问地址:
  * 本机: http://localhost:8080
  * 局域网: http://192.168.x.x:8080（实际 IP 会显示在界面上）

- 停止服务器:
  * 点击"停止服务"按钮，或
  * 关闭软件自动停止

================================================================================
五、网页查看器使用详解
================================================================================

5.1 桌面端界面
-------------
- 左侧: 平面图，显示所有标记点
- 右侧: 全景图，可鼠标拖拽旋转视角
- 顶部: 显示当前点位名称
- 导航: 上一张/下一张按钮

5.2 移动端界面
-------------
- 默认显示全屏全景图
- 右下角地图按钮: 显示平面图浮层
- 点击标记点切换
- 支持触摸拖拽和捏合缩放

5.3 操作
-----------
- 拖拽/滑动: 旋转视角
- 滚轮/捏合: 缩放
- 全屏按钮: 进入全屏模式
- 热点点击: 切换到对应点位

六、常见问题
================================================================================

Q1: 蓝牙连接失败怎么办？
A: 1. 确认相机已开机并进入蓝牙配对模式
   2. 在 iOS 设置中检查蓝牙权限
   3. 尝试关闭并重新打开 App
   4. 使用手动输入模式作为备用方案

Q2: 影像导入时显示"缺失"？
A: 1. 确认选择了正确的 SD 卡文件夹
   2. 检查文件名是否被修改过
   3. 使用"手动选择"功能指定正确的文件
   4. 确认文件在 SD 卡中存在

Q3: 局域网无法访问？
A: 1. 确保电脑和手机连接同一 Wi-Fi
   2. 检查防火墙是否阻止了端口
   3. 尝试更换端口（8080/8888/9000）
   4. 使用手机热点作为备用方案

Q4: 平面图显示模糊？
A: 1. 导入时选择高分辨率原图
   2. PDF 转换时会压缩到 2048px 宽，建议直接导出高清 JPG
   3. 在 PC 端可替换为更高清的平面图

Q5: 标记点位置不准确？
A: 1. 在 PC 端可以精确拖动调整位置
   2. 确保平面图未经过旋转或裁剪
   3. 坐标系统基于平面图左上角原点

================================================================================
七、注意事项
================================================================================

1. 数据安全
-----------
- 本软件 100% 离线运行，所有数据保存在本地
- 建议定期备份项目文件夹
- 导出项目时，照片会复制到项目目录，原文件保留

2. 文件命名
-----------
- 项目名称不要包含: \ / : * ? " < > |
- 相机文件名必须原样保存，区分大小写
- 建议不要手动重命名 SD 卡中的照片

3. 存储空间
-----------
- 影像文件占用空间较大，确保有足够存储
- 一个项目可能包含数百 MB 到数 GB 的照片
- 定期清理不需要的项目

4. 兼容性
---------
- iOS App 要求 iOS 16.0 或更高版本
- PC 管理器要求 Windows 10/11 64位
- 网页查看器支持现代浏览器（Chrome/Firefox/Safari/Edge）
